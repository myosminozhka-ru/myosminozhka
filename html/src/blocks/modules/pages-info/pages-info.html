Анимация точки

Функции:

app.cursorAnimation.onInit = () => {/*тут код*/} // стреляет при загрузке анимации
app.cursorAnimation.onAnimate = () => {/*тут код*/} // стреляет при движении мышью

Переменные:

app.cursorAnimation.count = 100; // (number) количество возможных кругов на экране
app.cursorAnimation.colors = [ '#FF4978', '#430A66', ]; // (array) массив цветов для градиента
app.cursorAnimation.radius = 40; // (number) радиус кругов
app.cursorAnimation.speed = 1; // (number) скорость исчезновение кругов
app.cursorAnimation.lifetime = 150; // (number) указано в ms, отсюда отнимается app.cursorAnimation.speed, при 0 круг исчезает
app.cursorAnimation.element = 'cursor'; (string) id элемента canvas
Два слова

Функции:

app.mainWords.onInit = () => {/*тут код*/} // стреляет при загрузке шапки

Переменные:

app.mainWords.x.position = 'left'; // (string) или 'right', добавляет активный класс соответствующему слову
app.mainWords.delay = 1; // (number) указано в s, задержка пока слова начнут ваыезжать
