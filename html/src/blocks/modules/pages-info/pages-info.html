Анимация точки

Функции:

app.cursorAnimation.onInit = () => {/*тут код*/} // стреляет при загрузке анимации
app.cursorAnimation.onAnimate = () => {/*тут код*/} // стреляет при движении мышью

Переменные:

app.cursorAnimation.params.count = 100; // (number) количество возможных кругов на экране
app.cursorAnimation.params.colors = [ '#FF4978', '#430A66', ]; // (array) массив цветов для градиента
app.cursorAnimation.params.radius = 40; // (number) радиус кругов
app.cursorAnimation.params.speed = 1; // (number) скорость исчезновение кругов
app.cursorAnimation.params.lifetime = 150; // (number) указано в ms, отсюда отнимается app.cursorAnimation.params.speed, при 0 круг исчезает
app.cursorAnimation.params.element = 'cursor'; (string) id элемента canvas
