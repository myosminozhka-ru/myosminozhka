    • Проблема

    «Строительный двор» на рынке с 1993 года, за это время у компании появилось множество конкурентов, которые представлены в Интернете, в том числе. «Строительный двор» предлагает огромный ассортимент товаров для строительства и ремонта, в этой нише много конкурентов, которые следуют современным тенденциям. Чтобы выделяться среди других представителей этой сферы, потребовался веб-ресурс с мощным SEO.

    Была решена проблема с продажами благодаря внедрению личного кабинета. до 80% пользователей покидают сайт, если не находят кнопку входа в личный кабинет. Мы сделали упор на юзабилити, поместив кнопку личного кабинета в шапке сайта.

    Описание проекта:
    Сфера деятельности заказчика – товары для ремонта и строительства
    Строительный двор – крупный интернет-магазин строительных товаров. Для сайта реализован современный функционал, дизайн сделан с акцентом на продукцию. У интернет-магазина мощное SEO, которое делает его лидером на рынке РФ.
    Решение
    • Разработали личный кабинет пользователя для удобного взаимодействия с сайтом, возможности управлять заказами и отслеживать их.
    • Сделали умный фильтр и поиск по товарам, чтобы легче было находить необходимый товар.
    • Интегрировали сайт с маркетплейсами, чтобы иметь возможность продвигать продукцию на других сайтах.
    • Сделали кастомную функцию наценки при выборе способа доставки, что позволяет при оформлении заказа сразу рассчитать стоимость доставки любым из выбранных способов.
    • Сделали современный дизайн с удобным юзабилити для комфортного использования сайта.
    • Сделали удобный каталог товаров, который разделен на категории и подкатегории, что упрощает поиск необходимых товаров.
    Особенности

    Кастомная наценка. При выборе одного из способов доставки к стоимости товара добавляется n-ая сумма. Например, при доставке курьером к товару прибавляется 2.8%.

    Разработали алгоритм рекомендаций, на основе которого формируются сопутствующие товары. Это происходит в автоматизированном режиме, исходя из предпочтений пользователей

    Результат
    Результатом проделанной работы стал полностью готовый к использованию сайт с обширным функционалом, в том числе удобным фильтром поиска, возможностью рассчитать стоимость доставки, интеграция с маркетплейсами и личный кабинет пользователя для управления заказами.
